BOM :- Brower Object Model

  • BOM is a browser-specific convention referring to all the objects exposed by the web browser. The BOM allows JavaScript to “interact with” the browser.
  • A window object is created automatically by the browser itself. Java Script’s window.screen object contains information about the user’s screen. It can also be written without the window prefix.

DOM :- Document Object Model

  • DOM is a programming interface for HTML and XML documents, that allows to create, manipulate, or delete the element from the document.
  • It defines the logical structure of documents and the way a document is accessed and manipulated. With the help of DOM, the webpage can be represented in a structured hierarchy.
